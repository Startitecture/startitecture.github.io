Compare<T>(T, T, Func<T, Object>[])

Performs a comparison of two objects of the same type and returns a value indicating whether one object is less than, equal to, or greater than the other.

Declaration
public static int Compare<T>(T valueA, T valueB, params Func<T, object>[] selectors)
Parameters
Type Name Description
T valueA

The first object to compare.
T valueB

The second object to compare.
System.Func<T, System.Object>[] selectors

The property selectors for the specified object.
Returns
Type Description
System.Int32

A signed integer that indicates the relative values of valueA and valueB, as shown in the following table.

Value Meaning

Less than zero valueA is less than valueB.

Zero valueA equals valueB.

Greater than zero valueA is greater than valueB.

Type Parameters
Name Description
T

The type of object to compare.
Remarks

This method uses System.Collections.Generic.Comparer`1 when no selectors have been specified. Calling this method from within an implementation of System.IComparable`1 on an object of the same type without specifying selectors will result in a recursive loop as System.Collections.Generic.Comparer`1 calls System.IComparable`1.CompareTo(`0).

Type Parameters
Name Description
T

The type of values to compare.
Remarks

This method uses System.Collections.Generic.EqualityComparer`1 when no selectors have been specified. Calling this method from within an implementation of System.IEquatable<T> on an object of the same type without specifying selectors will result in a recursive loop as System.Collections.Generic.EqualityComparer`1 calls System.IEquatable<T>.Equals(T). The same issue applies if the calling type is not System.IEquatable<T>.Equals(T).
